# CreditWise Loan Approval System

## Problem Statement
Develop a machine learning model to predict whether a loan application should be approved or rejected based on applicant financial and demographic data.

---

## Dataset
- Features include: Applicant Income, Credit Score, Loan Amount, DTI Ratio, Employment Status, etc.
- Target Variable: Loan Approval (Approved / Rejected)

---

## Tech Stack
- Python  
- pandas, NumPy  
- scikit-learn  
- Matplotlib, Seaborn  

---

## Approach

### 1. Data Preprocessing
- Handled missing values  
- Encoded categorical variables  
- Applied feature scaling  

### 2. Exploratory Data Analysis
- Correlation analysis  
- Feature distribution visualization  

### 3. Model Building
- KNN, Naive Bayes, Logistic Regression (final model)

### 4. Evaluation
- Precision: **90.69%**  
- Confusion Matrix for performance analysis  

---

## Results
- Achieved high precision to minimize incorrect loan approvals  
- Logistic Regression selected to reduce financial risk  

---

## Business Impact
- Reduces risk of approving high-risk applicants  
- Improves consistency and reliability of loan decisions
